////////////////////////////////////////////

工程是基于 vs2008下面编译的

////////////////////////////////////////////

开发者直接可以使用这3个对象
.\include ，
.\lib
.\bin , DLL 文件

////////////////////////////////////////////
.\DLL Debug   生成动态库，运行时库选择对应的 多线程调试 DLL
.\DLL Release 生成动态库，运行时库选择对应的 多线程 DLL


.\DLL DebugMT     生成动态库，运行时库选择对应的 多线程调试
.\DLL ReleaseMT   生成动态库，运行时库选择对应的 多线程


.\LIB Debug   生成静态库，是运行时库选择对应的 多线程调试 DLL
.\LIB Release 生成静态库，是运行时库选择对应的 多线程 DLL

.\LIB DebugMT     生成静态库，运行时库选择对应的 多线程调试
.\LIB ReleaseMT   生成静态库，运行时库选择对应的 多线程

//////////////////////////////////////////////////////////
工程编译完后，
1 所需的库 会自动copy到到 lib目录下
2 所需的头文件,会自动copy到到 include目录下
3 所需的DLL,会自动copy到到 bin目录下
